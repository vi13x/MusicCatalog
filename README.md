# Музыкальный каталог (Music Catalog)

Данный проект представляет собой разработку базового REST-сервиса для хранения и управления музыкальными альбомами.  
Сервис реализован с использованием **Spring Boot** и демонстрирует принципы построения многослойной архитектуры, обработку HTTP-запросов и использование DTO.

---


## Технологический стек

- **Java 21** — язык программирования  
- **Spring Boot 3.x** — основной фреймворк  
- **Spring Web** — создание REST API  
- **Maven** — система сборки и управления зависимостями  
- **Jackson** — сериализация и десериализация JSON  
- **Postman** — тестирование API  
- **Checkstyle** — контроль качества кода  

---
### Слои приложения

- **Controller** — обрабатывает HTTP-запросы и формирует HTTP-ответы  
- **Service** — содержит бизнес-логику приложения  
- **Repository** — отвечает за хранение и получение данных  
- **DTO** — используется для передачи данных клиенту  
- **Mapper** — преобразует Entity ↔ DTO  

Архитектура соответствует принципам **SOLID** и разделению ответственности (SRP).

---

## Модель данных

Основная сущность системы — **Album**.

Поля:

- `id` — уникальный идентификатор  
- `title` — название альбома  
- `artist` — исполнитель  
- `year` — год выпуска  

В текущей версии данные хранятся во временном in-memory хранилище (Map).
