# Музыкальный каталог

---

## Лакотко Владислав 405502

---

Данный проект является прототипом интернет-каталога по хранению и просмотру музыкальных композиций.

---

### Функционал проекта:

- Просмотр каталога музыкальных композиций;
- Поиск композиции по названию;
- Поиск по исполнителю;
- Просмотр информации о выбранной записи;

---

### Аналоги:

- Spotify
- Яндекс Музыка
- Apple Music
